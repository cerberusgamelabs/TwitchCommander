	╔═════════════════════════════╗
	║  Installation Instructions  ║
	╚═════════════════════════════╝

Step 1: Download Node.js from the following link:
	https://nodejs.org/en/download/current/
Step 2: Install as required.
Step 3: Run install.bat and follow instructions.
Step 4: Run the start.bat

When done, simply close the command prompt.

	╔═════════════════╗
	║ *!* WARNING *!* ║
	╚═════════════════╝

TwitchCommander now checks whether the configured target process is
running before sending commands. If the target game is not running,
commands will be skipped.

However, TwitchCommander does not currently verify that the game
window is the frontmost active window. If the target process is
running but another application is focused, automated inputs may be
sent to the wrong place.

Do not leave the bot running unattended unless you are comfortable
with that risk.
